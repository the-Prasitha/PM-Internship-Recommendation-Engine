# PM Internship Recommendation Engine

AI-powered internship recommendation engine that suggests internships based on user skills, interests, and experience using Full Stack and Machine Learning.

PROBLEM STATEMENT

Students often struggle to find internships that match their skills and interests.
Most internship platforms show thousands of listings without personalization.

This project builds an AI-powered recommendation engine that analyzes a student's
skills, interests, and experience to suggest the most relevant internships.

OBJECTIVE

The objective of this project is to build a full-stack web application that
uses machine learning to recommend internships to students.

The system analyzes user profiles and matches them with suitable internship
opportunities.

TARGET USERS

• College students
• Fresh graduates
• Internship seekers
• Career switchers

KEY FEATURES

• User profile input (skills, interests, experience)
• AI-based internship recommendation
• Skill matching algorithm
• Internship dataset analysis
• Resume-based recommendation (future feature)
• Clean web interface
